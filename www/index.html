Evothings Viewer


Info

This is the companion app for Evothings Studio.

Use this app with the Evothings Workbench desktop application to develop mobile apps for IoT in HTML5/JavaScript. The desktop application is available at evothings.com/download.

How to use

Launch the Evothings Workbench desktop application. Follow the on screen instructions to get a connect key.

Enter the connect key into this app and tap the Connect button.

Click the Run button of a project in the Workbench to launch an app on this device.

Learn more

For more information and tutorials, visit evothings.com.
